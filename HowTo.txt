
Build byu running one of below command

> cmake -E make_directory build

> cmake -E chdir build cmake .. -DCMAKE_EXPORT_COMPILE_COMMANDS=ON

> cmake --build build --target all --clean-first


Running Tests by executing one of following commands

> ctest build -V --build-and-test
> cmake --build build --target test -- ARGS="<ctest_args>"
> cmake -E env CTEST_OUTPUT_ON_FAILURE=TRUE cmake --build build --target test
